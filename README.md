# JavaScript Beginner Practice Problems

## Overview

This repository contains **50 beginner-friendly JavaScript practice problems** designed for students who have learned JavaScript fundamentals up to **arrays** and have not yet started learning **functions**.

The exercises are organized by topic and gradually increase in difficulty, helping learners build confidence in problem-solving and strengthen their understanding of core JavaScript concepts.

---

## Topics Covered

### Variables and Data Types
Learn the basics of:
- Declaring and initializing variables
- Working with strings and numbers
- Performing arithmetic operations
- Understanding different data types

### Input and Output
Practice:
- Receiving user input
- Displaying information
- Using input values in calculations and programs

### Conditional Statements
Develop skills in:
- Making decisions with `if`, `else if`, and `else`
- Comparing values
- Evaluating conditions
- Implementing simple business rules

### Logical Operators
Explore:
- Combining multiple conditions
- Using logical operators (`&&`, `||`, `!`)
- Building more advanced decision-making logic

### Loops
Learn how to:
- Repeat tasks efficiently
- Generate sequences of numbers
- Perform calculations using iteration
- Create simple patterns

### Nested Loops
Practice:
- Using loops within loops
- Generating structured outputs and patterns
- Building multiplication tables and grids

### Arrays
Understand:
- Creating arrays
- Accessing array elements
- Iterating through arrays
- Processing collections of data

---

## Challenge Problems

Additional challenge exercises are included to help learners:
- Strengthen problem-solving skills
- Apply multiple concepts together
- Improve logical thinking
- Prepare for more advanced JavaScript topics

## Prerequisites

Before attempting these exercises, learners should have a basic understanding of:

- Variables
- Data Types
- Operators
- User Input and Output
- Conditional Statements
- Loops
- Arrays

---

## Goal

The goal of this repository is to provide hands-on practice with JavaScript fundamentals before moving on to more advanced topics such as:

- Functions
- Objects
- DOM Manipulation
- Events
- APIs
- Advanced Problem Solving
